# remycord

A modern Discord TUI client built with Rust and Ratatui.

## Features

- **Fast & Lightweight** - Built with Rust for maximum performance
- **Keyboard-Driven** - Vim-style keybindings with full customization
- **Themeable** - Base16 theme support
- **Image Support** - Kitty graphics protocol for inline images
- **Secure** - OS-native credential storage
